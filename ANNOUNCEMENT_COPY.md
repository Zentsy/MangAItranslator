# MangAI Translator - Launch Copy

## Descricao curta do projeto

MangAI Translator e um app desktop para traducao assistida de manga e quadrinhos. Voce importa o capitulo, gera um rascunho por IA, revisa os blocos manualmente e exporta o resultado final.

## Texto para release no GitHub

### MangAI Translator - Beta

Primeiro beta publico do MangAI Translator.

O app foi feito para ajudar no fluxo de traducao e revisao de manga e quadrinhos:

- importar um capitulo inteiro por pasta ou pagina
- gerar rascunho com `Gemini` ou `Ollama`
- revisar bloco por bloco
- salvar o projeto localmente para continuar depois
- exportar em `.txt` e `.docx`

### Melhor forma de usar hoje

- `Gemini`: melhor experiencia para a maioria das pessoas
- `Ollama`: opcao local/offline, mas pode ficar mais lento dependendo da maquina

### O que ainda esta em beta

- refinamentos de interface
- polish de performance
- melhorias no fluxo local com Ollama

### Observacoes

Use o app apenas em materiais proprios, licenciados ou para os quais voce tenha permissao de localizacao/traducao.
